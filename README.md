# Mangify Frontend

## Technologies

-   TypeScript
-   React
-   React Router DOM
-   React Icons
-   SASS
-   axios
-   zustand
-   Vite

## Run frontend

1. Installation dependencies.

    For npm:

    ```shell
    npm install
    ```

    For yarn:

    ```shell
    yarn install
    ```

2. Start app

    For npm:

    ```shell
    npm run preview
    ```

    For yarn:

    ```shell
    yarn run preview
    ```
